## ✨ Features

- 🎨 **Color Picker** - Interactive color selection tool
- 🎯 **Palette Generator** - Create harmonious color schemes
- 💾 **Save Palettes** - Store your favorite combinations
- 📋 **Copy Color Codes** - Easy hex/RGB/HSL copying
- 🌓 **Light/Dark Mode** - Theme switcher
- 📱 **Responsive Design** - Works on all devices
- 🎭 **Multiple Color Models** - HEX, RGB, HSL support
- 📊 **Color Theory** - Learn about color harmony

## 🛠 Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Color Library:** Color.js / TinyColor2
- **Storage:** LocalStorage API
- **Styling:** CSS3 with Grid & Flexbox
- **Version Control:** Git & GitHub

## 📖 Usage Guide

### Color Picker
1. Click on the color input field
2. Select your desired color
3. View color codes in different formats
4. Copy the code to clipboard

### Palette Generator
1. Choose a base color
2. Select harmony type (Complementary, Analogous, Triadic, etc.)
3. Generate palette
4. Save for later use

### Manage Palettes
1. View all saved palettes
2. Edit existing combinations
3. Delete unwanted palettes
4. Export as JSON or image

## 🎨 Supported Color Formats

- **HEX** - #RRGGBB
- **RGB** - rgb(R, G, B)
- **HSL** - hsl(H, S%, L%)
- **HSV** - hsv(H, S%, V%)
- **Named Colors** - red, blue, etc.

---

## 🌟 Key Features

### Color Harmony Types
- **Complementary** - Opposite colors on wheel
- **Analogous** - Adjacent colors
- **Triadic** - Three colors equally spaced
- **Tetradic** - Four colors in rectangles
- **Split Complementary** - Two analogous to complement
- **Monochromatic** - Single hue variations

### Palette Management
- Create unlimited palettes
- Local storage persistence
- Export/Import functionality
- Share palettes with others

### Color Analysis
- Contrast ratio calculator
- Accessibility checker (WCAG)
- Color blindness simulator
- Color harmony visualizer
